WordPress Resturant Website

The objective of this project was to create a your own resturant and a website domain for it built using wordpress. Wordpress being completely new to me I had a hard time learning how to navigate through each function. It was easy to get stuck in one area, but after figuring out how to manage each element and how easily you are able to build off each component. I think my website ended up pretty well I know the image branding was consistent and carried along my fish and chips vibe.
